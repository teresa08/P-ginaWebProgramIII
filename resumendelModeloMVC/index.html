Resumen del Modelo MVC

Patrón de arquitectura para separar lógica, datos e interfaz de usuario.

¿Qué es el patrón MVC?

MVC (Modelo-Vista-Controlador) es un patrón de diseño arquitectónico que divide una aplicación en tres componentes interconectados, promoviendo la separación de responsabilidades.

Componentes del MVC

  • Modelo (Model): Gestiona los datos, la lógica de negocio y la persistencia (ej. base de datos, APIs, validaciones).
  • Vista (View): Representa la interfaz de usuario. Muestra los datos del modelo y envía acciones del usuario al controlador.
  • Controlador (Controller): Actúa como intermediario: recibe entradas del usuario (vía la vista), actualiza el modelo y decide qué vista mostrar.

Flujo típico de interacción

  1. El usuario interactúa con la Vista (ej. hace clic en un botón).
  2. La Vista notifica al Controlador.
  3. El Controlador actualiza el Modelo (ej. guarda datos).
  4. El Modelo notifica cambios (directa o indirectamente).
  5. El Controlador actualiza la Vista con los nuevos datos.

¿Por qué usar MVC?

  • Código más organizado y modular.
  • Fácil mantenimiento y pruebas unitarias.
  • Reutilización de modelos y vistas.
  • Equipos pueden trabajar en paralelo (front, back, lógica).
  • Escalabilidad a largo plazo.

Desventajas

  • Complejidad inicial para aplicaciones pequeñas.
  • Curva de aprendizaje para principiantes.
  • Sobrediseño si no se aplica con criterio.

Frameworks que usan MVC

  • Backend: Laravel (PHP), Ruby on Rails (Ruby), Spring MVC (Java), Django (aunque usa un patrón similar).
  • Frontend: Angular (con variantes), Backbone.js, Ember.js.

Comparación con otros patrones

  • MVP (Modelo-Vista-Presentador): El Presentador reemplaza al Controlador y tiene más control sobre la Vista.
  • MVVM (Modelo-Vista-ViewModel): Usa enlace de datos bidireccional (ej. Vue.js, Angular). La Vista se actualiza automáticamente.

Ejemplo conceptual (frontend puro)

Este ejemplo muestra una implementación mínima de MVC en el navegador, sin backend.

Modelo

// Almacena y gestiona los datos
const UserModel = {
  usuarios: [{ id: 1, nombre: "Ana" }, { id: 2, nombre: "Luis" }],
  agregar(nombre) {
    this.usuarios.push({ id: this.usuarios.length + 1, nombre });
  },
  obtenerTodos() {
    return this.usuarios;
  }
};

Vista

Controlador

const UserController = {
  init() {
    this.renderizar();
    document.getElementById("btnAgregar").onclick = () => {
      const nombre = document.getElementById("nombreInput").value.trim();
      if (nombre) {
        UserModel.agregar(nombre);
        this.renderizar();
        document.getElementById("nombreInput").value = "";
      }
    };
  },
  renderizar() {
    const lista = document.getElementById("lista");
    lista.innerHTML = UserModel.obtenerTodos()
      .map(u => `
• ${u.nombre}
`)
      .join("");
  }
};

document.addEventListener("DOMContentLoaded", () => UserController.init());
